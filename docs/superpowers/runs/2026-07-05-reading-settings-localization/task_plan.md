# 左下角阅读设置优化与中文化计划

## 目标

将文章页浮动阅读设置优化为中文化、可理解、位置稳定的左下角控制面板，保留现有字号和栏宽偏好持久化能力。

## 任务

- [x] RED：补充 `ReadingPreferences` 测试，覆盖中文面板标题、中文状态文案、按钮行为和左下角样式钩子。
- [x] GREEN：更新 `ReadingPreferences` 组件，让设置面板使用中文标签、状态值和更清晰的控制结构。
- [x] REFACTOR：更新 CSS，使面板固定在左下角，兼顾移动端与现有返回顶部按钮。
- [x] CHECK：运行组件测试、格式、lint，并用 `9856` 端口浏览器验证文章页。

## 验收标准

- 设置面板在文章页左下角显示，文案为中文。
- 字号按钮循环：标准字号 → 大字号 → 小字号 → 标准字号。
- 栏宽按钮循环：标准栏宽 → 宽栏 → 窄栏 → 标准栏宽。
- 偏好仍写入 `localStorage`，并应用到 `#article-content`。
- 不引入新英文 UI 文案。
